# Ruby-NFT

An NFT721 Contract for Ruby based on [MintChain](https://www.mintchain.io/)!

## Basic Usage

- Test on localhost
1. npx hardhat node
2. npx hardhat test --network localhost 

- Deploy to mint-sepolia
1. `npx hardhat run scripts/deploy.js --network mint-sepolia`
